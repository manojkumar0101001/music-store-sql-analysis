# 🎵 Music Store Data Analysis — SQL Project

## 📌 Project Overview
Analyzed a fictional music store database using SQL to answer 
business questions and uncover insights about customers, sales, 
and music genres.

## 🛠️ Tools Used
- MySQL
- MySQL Workbench
- Dataset: Kaggle (Chinook Music Store)

## 🗄️ Database Structure
11 Tables: artist, album, track, genre, media_type, 
employee, customer, invoice, invoice_line, playlist, playlist_track

## 📊 Analysis Levels

### 🟢 Easy Level
- Most senior employee
- Countries with most invoices
- Top 3 invoice totals
- Best city for promotional event
- Customer who spent the most

### 🟡 Moderate Level
- Rock music listeners (email list)
- Top 10 Rock artists by track count
- Tracks longer than average duration

### 🔴 Advanced Level
- Customer spending per artist (CTE)
- Most popular genre per country (Window Function)
- Top spending customer per country (Window Function)

## 💡 Key Insights
- 🇺🇸 USA has the most invoices (91) — top market
- 🏙️ Prague has highest revenue ($273) — best city for events
- 🎸 Rock is the #1 genre in almost every country
- 🎤 Led Zeppelin has most Rock tracks (114)
- 👑 Top customer spent $144.54 total
